# Create backend endpoint

Read:

- docs/plan.md
- .github/copilot-instructions.md

We are working only on the backend.

Task:
Create a minimal backend endpoint in TypeScript using Express.

Before editing files:

1. List the files you will change
2. Give the exact implementation plan
3. State any assumptions
4. Identify any packages that must already exist

Rules:

- Keep changes small and easy to review
- Do not add new libraries unless necessary
- Keep backend code under backend/src
- Prefer simple Express patterns
- Do not implement authentication, cloud storage, or database work unless explicitly requested

After approval:

- Implement only the agreed changes
- Explain how to test the endpoint locally
- Suggest a commit message
